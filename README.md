# Ratatouille

A technical challenge for NullCTF https://ctftime.org/event/2901, focusing on advanced malware analysis and threat hunting methodologies.

Engineered a realistic attack chain simulating an Advanced Persistent Threat (APT). Tasked competitors with analyzing and deobfuscating a complex, multi-stage PowerShell dropper. 

Designed the challenge to require deep reverse engineering skills, where players had to bypass obfuscation layers and trace the execution flow to uncover the final Remote Access Trojan (RAT) payload. 

Showcased practical expertise in malware development techniques, script obfuscation, and blue-team forensic analysis.
